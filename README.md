# Calculate First/Follow Set of given grammar

## Notice
The project is in development.

## Usage
### Install
```
git clone https://github.com/ftfuntjh/First-Follow.git
cd first-Follow
mkdir build && cd build
cmake .. && cmake -- build .
./build/lr1
```

### Example

The example grammar is
```
E -> TE_
E_ -> +TE_
E_ -> ^
T -> FT_
T_ -> *FT_
T_ -> ^
F -> (E)
F -> i
```
**the '^' means 'Îµ'**

### the result is
```
--- the first set is ---
E -> {'(' ,'i'}
E_ -> {'+' ,'^'}
F -> {'(' ,'i'}
T -> {'(' ,'i'}
T_ -> {'*' ,'^'}
--- the follow set is ---
E -> {'$' ,')'}
E_ -> {'$' ,')'}
F -> {'$' ,')' ,'*' ,'+'}
T -> {'$' ,')' ,'+'}
T_ -> {'$' ,')' ,'+'}
```
